## ðŸ§© System Workflow
1. **Product NFT Creation**  
   Every new product is minted on the blockchain as a unique identifier (NFT or unique ID).

2. **Lifecycle Updates Recorded On-Chain**  
   Each supply chain operation is signed and stored immutably:
   - Manufacturing  
   - Warehousing  
   - Transportation  
   - Shipping  
   - Customer Delivery

3. **Immutable Ledger**  
   No recorded data can be altered or deleted.

4. **Customer Verification**  
   When a user scans the QR code on the product, they can view:
   - Production details  
   - Storage and transport history  
   - Quality checks  
   - Delivery logs

   All data is verified on-chain in real time.
